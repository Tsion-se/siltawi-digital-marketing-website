## 📌 Project Overview

Siltawi Digital Marketing Website is a modern, responsive company profile website developed for **Siltawi Digital Marketing**, a digital marketing and creative agency based in Ethiopia.

The website is designed to showcase services, portfolio, team members, testimonials, and contact information in a professional, high-conversion digital format.

It focuses on:
- Modern UI/UX design
- Mobile-first responsiveness
- Smooth animations
- SEO optimization
- Clean and scalable code structure
